**www.leeyuchen.top**  
是一个集 **私人主页、加密访问、在线工具等...** 于一体的个人网站仓库

## ✨ 核心特性
- ✅ **AES-256 + SHA-256 加密主页**，隐私安全
- ✅ **内置翻译工具**（类 Bing 翻译，代码块风格）
- ✅ **工具集合页面**（加密、Base64、JSON、时间戳…）
- ✅ **渐变毛玻璃 UI**，暗黑/浅色自动适配
- ✅ **响应式布局**，手机/电脑完美显示
